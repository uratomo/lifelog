- Udemy react-redux セクション 4 (redux について)
  - redux
    - react の欠点である子コンポーネントへの props のバケツリレーをしなくてよくなる
    - state の代わりに store の値を全コンポーネントで共有できる

    - 結論: react 本体では、JSX 以外での記述量を減らすこともできる。
